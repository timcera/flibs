Name

flibs/strings - Handle a list of tokens

Synopsis

  • use tokenlists

Description

The tokenlists module builds on the tokenize module and provides an object-oriented interface to the set of tokens that was found in a string. The object holds a copy of the string, so that parsing is done only once and you can randomly access the tokens.

INTERFACE

The module defines a single class, tokenlist and passes on the predefined character classes from the tokenize module.

use tokenlists

To import the definitions, use this module.

type(tokenlist)

The derived type with methods for handling a list of tokens.

call list%set_tokenizer( gaps, separators, delimiters )

Initialise the tokenizer characteristics for the tokenlist object. (See tokenize module for details).

class(tokenlist) list

The tokenlist object to be configured.

character(len=*) gaps

The string of characters that are to be treated as "gaps". They take precedence over the "separators". Use "token_empty" if there are none.

character(len=*) separators

The string of characters that are to be treated as "separators". Use "token_empty" if there are none.

character(len=*) delimiters

The string of characters that are to be treated as "delimiters". Use "token_empty" if there are none.

call list%tokenize( string )

Tokenize the string and store the results for later retrieval. A copy of the string is stored in the tokenlist object.

class(tokenizer) list

The tokenlist object to be used

character(len=*) string

The string to be split into tokens.

number_tokens = list%number()

Get the number of tokens that was found in the string. one.

type(tokenlist) list

The tokenlist object to be used

length_token = list%length(idx)

Get the length of the idx'th token.

type(tokenlist) list

The tokenlist object to be used

integer idx

The index of the token you want to know the length of

substring = list%token(idx)

Get the idx'th token. The returned string is exactly of the length of the token.

type(tokenlist) list

The tokenlist object to be used

integer idx

The index of the token you want returned

Convenient parameters:

  • token_whitespace - whitespace (a single character)

  • token_tsv - tab, useful for tab-separated values files

  • token_csv - comma, useful for comma-separated values files

  • token_quotes - single and double quotes, commonly used delimiters

  • token_empty - empty string, useful to suppress any of the arguments in the set_tokenizer routine.
